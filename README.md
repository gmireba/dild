### Описание
Проект **Training Planner** — это консольное приложение (или скрипт) на языке Python, предназначенное для планирования тренировочного процесса. Программа помогает структурировать занятия, отслеживать прогресс и организовывать тренировки в удобном формате.

Основные возможности:
*   Управление списком упражнений.
*   Сохранение планов тренировок.

### Как запустить
Для работы программы необходим Python версии 3.x.

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/demidov.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd training_planner
   ```
3. Запустите программу:
   ```bash
   python3 main.py
   ```

---

### Примеры использования
После запуска программы следуйте инструкциям в консоли. 

**Пример сценария (тест):**
1. При запуске программа запрашивает тип тренировки (например, "Силовая").
2. Пользователь вводит список упражнений: "Приседания, Жим лежа".

3. Программа подтверждает сохранение плана и выводит итоговый график на неделю.
